# ğŸ‘¥ User Roles

Role-based access control ensures that each user interacts only with the features relevant to their responsibilities.

#### **Admin**
Full access to all modules and features.  
Can manage users, system settings, menus, dishes, orders, suppliers, and inventory.

#### **Waiter**
Responsible for handling customer orders.  
Can create new orders, update order status, view menus and dishes, and follow up on active tables.

#### **Chef**
Focused on kitchen workflow.  
Can view incoming orders, update preparation statuses, and access dish descriptions and ingredients.
